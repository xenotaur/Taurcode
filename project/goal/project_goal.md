# Project Goal

## Objective
Create a toolkit to manage AI coding prompts with various text-replacement utilities.

## Intended Outcome
- A maintainable set of AI coding prompts that can be reused across workflows.
- Packaging and formatting support for text expansion/injection systems (Espanso is currently present).
- Clear operating guidance so contributors can add or refine prompts safely.

## Intended Users / Stakeholders
- Developers using AI assistants for coding and code review.
- Maintainers curating prompt libraries and replacement packages.

## In Scope
- Prompt content curation for coding workflows.
- Text-replacement packaging and trigger definitions.
- Documentation describing usage and repository conventions.

## Out of Scope (Initial)
- Building or maintaining a full standalone IDE plugin ecosystem.
- Claims of support for injection systems not evidenced in-repo.
- Runtime telemetry/analytics systems.

## Success Direction
- Repository provides practical prompts that are easy to trigger and adapt.
- Contributors can add/update prompt utilities with clear process and low ambiguity.
- Packaging artifacts remain coherent and usable for documented systems.
