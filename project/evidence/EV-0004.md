# Evidence Record EV-0004

## Scope

This record summarizes closeout evidence that the Taurcode Espanso roundtrip foundation
is complete for the current v1 support boundary.

## Observed Implementation Signals

- Canonical Markdown prompt files under `prompts/taurcode/` are the source of truth
  for the Taurcode prompt package.
- The CLI validates canonical prompts and exports supported prompts to Espanso package output.
- Espanso import and merge import support static replacements while preserving curated
  prompt metadata where Espanso does not own a field.
- Unsupported Espanso constructs are diagnosed and preserved through raw fallback
  behavior instead of being silently converted into supported prompts.
- Espanso package metadata assets roundtrip through `prompts/<package>/espanso/` and
  generated package output.
- Semantic normalization and `taurcode roundtrip espanso` compare supported prompt and
  package semantics rather than byte-for-byte YAML formatting.
- Espanso lint/preflight diagnostics, prompt-source linting, prompt formatting,
  operational scripts, tests, and coverage command wiring are present.
- Documentation is organized under the Diátaxis-style `docs/` index and includes
  canonical prompt format, Espanso integration, import/debug workflows, roundtrip
  fidelity checks, tutorials, and explanations.

## Closeout Validation Path

The foundation should continue to pass this local smoke sequence:

```bash
scripts/develop
scripts/lint
scripts/format
scripts/test
scripts/coverage
taurcode validate --prompts prompts/taurcode
taurcode export espanso --prompts prompts/taurcode --output tmp/exported/taurcode
taurcode roundtrip espanso --input tmp/exported/taurcode --prompts prompts/taurcode
```

When the `taurcode` console script is unavailable because local development
installation failed, the equivalent module invocation is:

```bash
PYTHONPATH=src python -m taurcode.cli validate --prompts prompts/taurcode
PYTHONPATH=src python -m taurcode.cli export espanso --prompts prompts/taurcode --output tmp/exported/taurcode
PYTHONPATH=src python -m taurcode.cli roundtrip espanso --input tmp/exported/taurcode --prompts prompts/taurcode
```
