# Evidence Record EV-0003

## Scope

This record summarizes implementation evidence for the adopted frontmatter dependency audit and shim-removal design proposal.

## Observed Implementation Signals

- The audit identified that Taurcode had a local top-level `src/frontmatter` shim that could shadow the declared `python-frontmatter` dependency.
- The audit found the local shim lacked full YAML semantics compared with `python-frontmatter` and PyYAML behavior.
- The implementation removed the local top-level `frontmatter` shim so Taurcode no longer ships that shadowing package.
- Prompt loading now relies on the declared `python-frontmatter` runtime dependency.
- Packaging discovery was constrained so Taurcode does not accidentally ship a top-level `frontmatter` package.
- Prompt-loader and validation tests were added or updated for scalar YAML metadata, quoted strings, YAML comments, nested `targets`, malformed frontmatter errors, body normalization, and shim removal/import resolution.
