# Taurcode

Espanso prompt package for Taurcode development workflows.

This package contains text-expansion prompts used for planning, designing,
debugging, reviewing, and generating structured Codex Cloud prompts.

For canonical prompt authoring and review guidance, see the source repository's [prompting best practices guide](https://github.com/xenotaur/Taurcode/blob/main/docs/how-to/best-practices/prompting-best-practices.md).
