Evaluate this audit report

Are the issues raised audit report valid, feasible, and consistent with the project’s goals, and actionable?

Does this report raise valid issues we should address? If it is not valid, say why not and stop.

If this report raises valid issues, are they feasible to address? If it is not feasible, say why not and stop.

If this report raises valid and feasible issues to address, are they consistent with the project goals? If not consistent, say why and stop.

If this report raises valid, feasible, and goal-consistent issues, are they actionable? If not actionable right now, say why and stop.

If the report raises valid, feasible, and goal-consistent issues that are actionable, please present a plan to address them. Review the work to be done, review recommended best practices in the area, consider the pros and cons, generate a set of potential options, evaluate them with respect to the best practices. 

If key information is missing needed for planning, stop and ask for clarification. 

If there is a serious issue that needs to be addressed in the audit before planning, also stop and alert the user. 

Otherwise, if there is enough information to proceed, outline a set of recommendations to address the issue respecting the user’s intent in a best-practices way. 

Let’s think step by step and check our work to make sure we get the right answer, and cite or quote reputable sources in our answer so we know it is grounded.
