Please present a plan to accomplish this task.

Review the work to be done, review recommended best practices in
the area, consider the pros and cons, generate a set of potential
options, evaluate them with respect to the best practices.

If key information is missing, stop and ask for clarification.

If there is a serious issue that needs to be addressed, also stop
and alert the user.

Otherwise, if there is enough information to proceed, outline a
set of recommendations to accomplish the task respecting the user’s
intent in a best-practices way.

Let’s think step by step and check our work to make sure we get
the right answer, and cite or quote reputable sources in our
answer so we know it is grounded.
